about

I’m Chaudhary Mohd Shoeb, a self-taught front-end developer with a passion for building responsive and user-friendly web applications. My journey in web development started with curiosity, and through dedication, I have honed my skills in HTML, CSS, JavaScript, and frameworks like Tailwind. I love tackling challenges that improve user experiences and make the web more accessible.

My recent projects include developing interactive websites and applications like Tourrr, SidcupFamilyGolf , and a Rock, Paper, Scissors game with a custom UI. I'm constantly expanding my knowledge, learning new tools, and improving my craft.

I'm passionate about creating clean, efficient code that not only looks great but also enhances functionality and performance. With an eye for detail and a drive for continual learning, I enjoy exploring new technologies and methodologies to stay up-to-date in the fast-evolving world of web development. My problem-solving mindset allows me to break down complex challenges into simple, elegant solutions. As I grow, I aim to continuously refine my front-end expertise to build innovative, impactful digital experiences.
